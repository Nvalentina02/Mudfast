# MUDFAST - Plataforma de Mudanzas

Aplicación móvil React Native con Expo para gestión de servicios de mudanza.

## Instalación

1. Instalar dependencias:
```bash
npm install
```

2. Iniciar la aplicación:
```bash
npx expo start
```

## Estructura del Proyecto

- `app/` - Pantallas de la aplicación con Expo Router
- `assets/` - Imágenes y recursos estáticos
- `public/` - Archivos públicos

## Pantallas Disponibles

### Usuario
- Inicio y autenticación
- Solicitar mudanza
- Ver cotización
- Perfil del proveedor
- Sistema de pagos
- Historial de mudanzas
- Centro de ayuda

### Proveedor
- Panel de servicios
- Gestión de solicitudes
- Historial de servicios
- Centro de ayuda

## Tecnologías

- React Native
- Expo
- Expo Router
- JavaScript (JSX)
